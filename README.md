#과제5). 배열 

1). 배열 알아보기

    과제에 제시한 대로 
    *처음부터 끝까지
    *뒤에서부터 처음까지
    *배열 전체의 총합, 최대값, 최소값을 출력해봤습니다.
    srand(time(NULL))을 사용하여 매 실행마다 새로운 무작위 값을 생성합니다.
    
2). 홀수 마방진

    마방진을 만드는 알고리즘을 학습하여 그 과정을 코드로 구현했습니다.
    2차원 배열을 생성하고, 정수를 사용하여 배열 전체를 채웠습니다.
    5*5 배열이라서 가로 5 세트, 세로 5 세트, 대각선 방향별 하나가 되어 총 12 세트를 출력했습니다.
